# PartyBuffs
Cycles through party members and casts buffs based on their class and yours.


        PartyBuffs Commands:
        /pbuff - Buff next party member
        /pbuff reset - Reset to player
        /pbuff all - Start buffing from player
        /pbuff status - Show current status
        /pbuff show - Show the button
        /pbuff hide - Hide the button
        
        Supported Classes:
        Paladin: Casts Blessing of Might on Warriors
        Mage: Casts Arcane Intellect on everyone
